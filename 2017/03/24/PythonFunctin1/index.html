Python类常用的内置方法

發表於 2017-03-24   |   分類於 python
  • init(self):初始化对象，在创建新对象时调用,这个方法可以用来对你的对象做一些你希望的初始化

  • new(cls,*args,**kwd) 实例的生成操作.在init()之前被调用，用于生成实例对象。利用这个方法和类属性的特性可以实现设计模式中的单例模式，代码如下：

    class Singleton(object):
 __instance = None

    def __init__(self):
        pass

    def __new__(cls,*args,**kwd):
        if Singleton.__instance is None:
            Singleton.__instance = object.__new__(cls,*args,**kwd)
        return Singleton.__instance

if __name__ == '__main__':
s1 = Singleton()
s2 = Singleton()
print("s1 id %s ,s2 id %s " % (id(s1),id(s2)))
  • getattr(self,name) 获取属性的值，当读取对象的某个属性时，python会自动调用getattr()方法
  • setattr(self, name):设置属性值，当使用赋值语句对属性进行设置时，python会自动调用setattr()方法
  • getattribute(self, name)：与getattr(self,name)类似,如果定义了getattr(self, name),则先调用getattr。
  • getitem(self,key) 获取序列的索引key对应的值，等价于seq[key]，详细用法

  • str():str()用于表示对象代表的含义，返回一个字符串;默认输出对象，可以对其进行定制化，输出想要的东西。示例代码如下：

    class Fruit(object):
"""docstring for Fruit"""

    def __init__(self, color="red", price=0):
        self.color = color
        self.price = price

    def __str__(self):
        return self.__doc__

if __name__ == '__main__':
    f1 = Fruit("blue", 1)
    print(f1) # 输出docstring for Fruit
