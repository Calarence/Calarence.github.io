python内建方法

發表於 2017-03-24   |   分類於 python

python 版本：2.7.13

方法名称 用法解释 其他
abs（x） 返回x的绝对值
all(iterable) 如果iterable为空或者iterable内所有元素为真返回True，否则返回False
any(iterable) 如果iterable中有元素为True,则返回True；如果iterable为空返回false
basestring() 用了判断一个对象是否是str的实例 python3中已经除去了该方法
bin（x） 将整数转换成二进制字符串。 如果x不是一个int对象，则必须定义index()方法并返回一个整数
bool(x) 如果x为false,或者为空，返回False；否则返回True
callable(object) 如果object是可调用的返回True，否则返回False。如果返回True的话，有可能调用失败；但是返回False的话，一定不会调用成功 如果一个类返回一个实例，那么它就是可调用的；如果类实例含有call()方法，那么它也是可调用的
chr() 返回ASCII码为整数的字符 参数取值范围为0~255，如果不在这个范围，就会报ValueError
classmethod()
cmp(x,y) 比较x与y的大小
compile
dict 创建字典类型 参数可以为mapping类型或iterable类型
complex() 用于创建一个复数 加减号两侧不能包含空格，否则会报ValueError
delattr（object，name） 用来删除一个对象的属性，
dir（object） 如果参数为空的话，返回当前作用域的name列表，如果指定参数的话，返回该对象的有效属性列表
divmod(x,y) 返回x除以y的商和余数
enumerate(sequence, start=0) 返回一个枚举对象，参数必须是序列、迭代器或其他支持迭代的对象；enumerate()返回迭代器的next方法，next方法的每次调用返回一个元组，元组中包括一个计数（默认从0开始）和从sequence中迭代而来的值
eval(expression[, globals[, locals]]) expression参数当作python 表达式解析并执行，使用global指定全局变量，使用locals指定局部变量
file（name） file类型的构造函数;当打开一个文件时更倾向于使用open()函数；file()函数更适用于类型判读，如isinstance(f,file)
filter(function, iterable) 返回一个列表，列表中元素为iterable中function条件为真的那些 等价于[item for item in iterable if function(item)]，注意python2与python3返回值不同，python2返回list，而python3返回filter对象
float(x) 将数字或字符串转换为浮点数
format(value[, format_spec]) 将value转换为格式化表示
frozenset([iterable]) 返回frozenset对象，set的一种类型 frozenset是冻结的集合，它是不可变的，存在哈希值，好处是它可以作为字典的key，也可以作为其它集合的元素。缺点是一旦创建便不能更改，没有add，remove方法
getattr(object, name[, default]) 返回object的name属性的值，name必须为一个字符串，如果name为object对象的一个属性，返回的结果为该属性的值
globals() 返回一个字典，代表当前全局变量 globals(x)，x成为全局变量
hasattr(object, name) 参数为对象和字符串，如果name为object的一个属性则返回True，否则返回False
hash(object) 返回object的hash值。用来在字典查询时快速比较键值，数字值相同的话有相同的hash值，如1跟1.0的哈希值相同
help([object]) 调用系统内建的帮助系统
hex(x) 将一个整数转换为以‘0x’开头的小写十六进制 如果x不是python的int或long对象，那么它必须定义hex()方法来返回一个字符串
id(object) 返回对象的标识符。这是一个整数，在对象生命周期内，来保证它的唯一性，并且不可变 Two objects with non-overlapping lifetimes may have the same id() value；这是对象在内存中的地址
input 获取用户输入
