###集合的接口与具体实现
java中集合的接口与实现是分离的。
java接口如下图

  • List：元素有序
  • Set：元素不允许重复。两个元素的hashCode如果一致，则两元素相同
  • SortedSet、SortedMap：待补充
  • NavigableSet、NavigableMap：待补充

###链表(Java中所有链表都是双向链表）

  • 链表添加元素
  • 链表删除元素：通过迭代器的remove方法
    先添加三个元素，然后将第二个元素删除
    List staff = new LinkedList<>();
    staff.add(“Amy”);
    staff.add(“bob”);
    staff.add(“clarence”);
    Iterator iter = staff.iterator();
    String first = (String) iter.next();
    String seconde = (String) iter.next();
    iter.remove();//删除上一个元素
  • 链表在指定位置添加元素：通过迭代器（listiterator）add方法（示例代码）
    先添加三个元素，然后在第一个后面插入一个元素
    List staff = new LinkedList<>();
    staff.add(“Amy”);
    staff.add(“bob”);
    staff.add(“clarence”);
    ListIterator iter = staff.listIterator();
    iter.next();
    iter.add(“juliet”);

###数组列表–ArrayList

  • ArrayList封装了一个动态再分配的对象数组，其实现细节是？
  • ArrayList与Vector的区别：Vector类的所有方法都是同步的。可以由两个线程安全地访问一个Vector对象。但是，如果由一个线程访问Vector，代码要在同步操作上耗费大量的时间。这种情况是很常见的。而ArrayList方法不是同步的，因此，建议在不需要同步时使用ArrayList，而不要使用Vector
