Java基础

發表於 2017-11-28   |   分類於 blog

Java基础回顾

  • 大小写敏感
  • 4个访问修饰符及范围
  • 类的命名规范：类名是以大写字母开头。驼峰命名
  • 源代码的文件名必须与public类的名字相同，并以.java作为扩展名
  • 使用javac xxx.java编译程序，使用java xxx来运行程序。xxx这里值类名
  • 注释：有三种
    • 行注释: //something
    • 段注释： / something /
    • 可自动生成文档的注释： /* something /
  • 基本数据类型：共8种：4种整型，2种浮点型，1种字符型，1种布尔型
    • 整型：范围从小到大依次为：byte、short、int、long，所需要的存储空间依次为1字节、2字节、4字节、8字节。java中所有数值类型所占据的字节数量与平台无关
    • 浮点型：float、double，double的精度是float类型的两倍，也称为双精度。所需存储空间以此为4字节、8字节
    • 字符类型：char。编码格式的相关内容
    • 布尔类型：boolean。整数值和布尔值之间不能进行相互转换
  • 引用类型：数组、类、接口
  • 变量：变量的声明以及命名规范；变量的初始化；在java中，变量的声明尽可能地靠近变量第一次使用的地方，这是一种良好的程序编写风格
  • 常量：用关键字final修饰常量，表示该变量智能被赋值一次。一旦被赋值之后，就不能被修改了。常量名使用全大写。如果希望某个常量可以在一个类中的多个方法中使用，通常将这些变量称为类常量，使用static final来设置一个类常量。类常量的定义位于main方法外部
  • 运算符：+，-，*，/。当参与/运算的两个操作数都是整数时，表示整数除法，否则表示浮点除法。整数被0整除会产生一个异常，而浮点数被0整除将会得到无穷大或NaN结果
  • 数值类型之间的转换：实心箭头表示无信息丢失的转换；虚箭头表示可能有精度损失的转换。

    n+f,n是整数，f是浮点数，需要将两个操作数转换为同一种类型，再进行运算。
    • 如果两个操作数中有一个是double类型，另一个操作数就会转换为double类型
    • 如果一个操作数是float类型，另一个操作数机就会转换为float类型
    • 如果一个操作数是long类型，另一个操作数就会转换为long类型
    • 否则，两个操作数都将转换为int类型。
  • 强制类型转换
  • 流程控制
    • 块作用域：块决定了变量的作用域；不能在嵌套的两个块中声明同名的变量
    • if else
    • while、do…while
    • for循环：
    • switch case:case标签可以是char/byte/short/int的常量表达式；或枚举常量；从java SE7开始，case标签还可以是字符串字面量
    • 中断控制流程语句：break，continue
  • 字符串
  • 大数值：BigInteger和BigDecimal：这两个类可以处理包含任意长度数字序列的数值
  • 数组
    • 数组拷贝：使用Arrays类的copyOf方法会将一个数组的所有值copy到一个新的数组中去，这时候对新数组的修改将不会影响到原先数组。
