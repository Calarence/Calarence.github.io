nodejs进阶一

發表於 2015-11-22   |   分類於 MEAN框架学习
  • javascript语法、特性学习，可参考js编程全栈
  • node特点：
    • 异步I/O
    • 事件与回调函数
    • 单线程
  • node的模块–modole:ConnmonJS对模块的定义十分简单，主要分为模块引用、模块定义和模块标识三个部分。
    • 模块引用：示例代码var math = require('math')在CommonJS规范中，存在require()方法，这个方法接收模块标识，以此引入一个模块的api到当前上下文中
    • 模块的定义：在模块中，上下文提供require()方法来引入外部模块。对应引入功能，上下文提供了exports对象用于导出当前模块的方法或者变量，并且它是唯一导出的出口。在模块中，还存在一个module对象，它代表模块自身，而exports是module的属性。在Node中，一个文件就是一个模块，将方法挂载在exports对象上作为属性即可定义导出的方式。
      //math.js exports.add = function(){ var sum =0, i = 0, args = arguments, l = args.length; while(i<1){ sum += args[i++]; } return sum; };
    • 在另一个文件中，可通过require()方法引入模块后，就能调用定义的属性或方法了：
      //programs.js var math = require('math'); exports.increment = function(val){ return math.add(val,1); };
      这篇文章详细讲述了module.exports和module
