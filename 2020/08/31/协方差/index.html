协方差

發表於 2020-08-31   |   分類於 blog

协方差的概念

协方差是描述两个变量之间相互关联程度的一个特征。定义如下：

设(X,Y)是一个二维随机变量，若E((X-E(x))(Y-E(Y)))存在，则称此期望为X与Y的协方差。记为Cov(X,Y) = E((X-E(x))(Y-E(Y)))。特别有Cov(X,X) = Var(x)。

X的方差是X-E(X)与X-E(X)乘积的期望，如今把一个X-E(X)换成Y-E(Y);又有X，Y二者的参与，由此得出协方差的名称。

根据协方差的定义，我们可以知道Cov(X,Y)可正可负，也可为零。

  • 当Cov(X,Y)>0时，称X与Y正相关。这时两个偏差(X-E(X))与(Y-E(Y))同时增加或同时减少。由于E(X)、E(Y)都是常数，故等价于X与Y同时增加或同时减少，这就是正相关的含义
  • 当Cov(X,Y)<0时，称X与Y负相关，这时X增加而Y减少，或Y增加而X减少，这就是负相关的含义。
  • 当Cov(X,Y) = 0时，称X与Y不相关。

协方差的性质

协方差的性质

1、Cov(X,Y) = E(XY) - E(X)E(Y)

证明：

$$Cov(X,Y) = E((X-E(X))(Y-E(Y))) = E(XY-XE(Y)-Y(EX)+E(X)E(Y)) = E(XY) -E(XE(Y)) - E(YE(X)) = E(XY) -E(Y)E(X)-E(X)E(Y) = E(XY) -E(X)E(Y)$$

要注意E(X)与E(Y)是常数。

2、若X，Y相互独立，则Cov(X,Y)为0。

这个比较好证明，因为当X,Y相互独立时，E(XY)等于E(X),E(Y).

但这个性质反过来不成立，即Cov(X,Y)等于0时，并不能得到X，Y相互独立。

设随机变量$X～N(0,\sigma^2),且令Y=X^2$,则X与Y不独立。但此时的协方差为$Cov(X,Y) = Cov(X,X^2) = E(X*X^2)- E(X)E(X^2) = 0$。

总结一下，Cov(X,Y)为零时，可以得出两者不相关，但并不能确定两者是否独立。这个例子说明，独立必定导致不相关，但不相关不一定导致独立，独立是不相关的一个子集。

3、任意随机变量X与常数a的协方差为零，即Cov(X,a) = 0

4、对于任意常数a，b有Cov(aX,bY) = abCov(X,Y)

5、对于任意二维随机变量(X,Y),有Var(X+Y) = Var(X) + Var(Y) + 2Cov(X,Y)

同理，Var(X-Y) = Var(X) + Var(Y) - 2Cov(X,Y)

6、设X,Y,Z是任意三个随机变量，则Cov(X+Y,Z) = Cov(X,Z) + Cov(Y,Z)

上式的证明用到了Cov(X,Y) = E(XY) - E(X)E(Y)这一性质。
