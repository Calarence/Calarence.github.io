事件的概率

發表於 2020-08-25   |   分類於 blog

概率和事件

概率：是表示某种情况或事件出现的可能性大小一种数量指标，它介于0与1之间。

事件：某种或某些情况的陈述，它可能发生，也可能不发生，发生与否，要等到有关的试验有了结果以后才能知晓，它只是一种陈述。例如，事件A：下午六点前不会下雨。

事件有如下几个特征：

  1. 它是一种观察
  2. 观察的全部结果在试验前是明确的，如下雨或不下雨，就两种结果；投硬币也只有两种结果；事件的结果也有可能是连续值，如降水量
  3. 陈述确定了试验的全部结果中的一部分，这个确定的一部分叫做一个事件。如E1={掷出偶数点}=(2,4,6) E2={掷出奇数点}=(1,3,5), E1,E2只是全部试验结果的集合(1,2…6)的一部分

事件根据可能性的大小分为：不可能事件、随机事件和必然事件。

古典概率

古典概率是指每个事件发生的可能性都相同，如掷骰子，每个点出现的概率都相等。

古典概率计算公式：设一个试验有N个等可能的结果，而事件E恰包含其中的M个结果，则事件E的概率，记为P(E),定义为$P(E) = \frac {M}{N}$。每个结果的概率为$\frac{1}{N}$, 而事件E包含M个结果，其概率理应为$\frac{1}{N}$的M倍，即$\frac{M}{N}$

局限性：只能用于全部试验结果为有限个，且等可能性成立的情况。

几何概率

适用于结果为无限个；并且基于几何图形的面积、体积、长度而算出的。

例：若甲、乙两人约定1点到2点之间在某吃碰头，约定先到者等候10分钟即可离去，假设甲、乙两人随意在1点到2点之间到达该处，问甲、乙两人碰上的概率有多大。因为10分钟内可以无限划分，即结果是无限的。这类问题可以转换为求面积来计算。

古典概率的排列组合公式

  1. n个相异物件取r(1≤r≤n)的不同排列总数，则$P^n_r = n(n-1)(n-2)…(n-r+1)$; 当n=r时，有 $P^n_r = n (n-1)(n-2)…1 = n!$ 。 从 a,b,c,d 这4个字母中任意取2个做排列，$P^n_r = 4 * (4-2+1) = 12$

  2. n个相异物件取r个的不同组合数，则$C^n_r = \frac{P^n_r}{r!} = \frac{n!}{r!(n-r)!}$。每一个包含r个物件的组合可以产生$r!$个不同的排列，故排列数应为组合数的$r!$倍。如从a,b,c,d 这4个字母中任意取2个做组合，则有$C^4_2 = \frac{4!}{2!2!} = \frac{432}{2*2} = 6$

  3. n个相异物件分成k堆，各堆物件数为$r_1,r_2….r_k$ 的分法是$\frac{n!}{r_1! r_2!…r_k!}$

事件的蕴含、包含及相等

若A发生时，B必然发生，则A蕴含B或者B包含A，记为$A \subset B$或$B \supset A$。 若$A \subset B$ 并且$B \subset A$，则有$A = B$。

例. 若A={掷出的点数之和大于10}，B={至少有一粒骰子掷出6}，则有$A \subset B$

事件的互斥和对立

互斥：若两事件A，B不能在同一试验中都发生，但都可以不发生，则称两者是互斥的。互斥事件是说构成这两个事件的试验结果不能有公共的。如A={掷出点数为4的倍数}，B={掷出的点数为3的倍数}，A和B为互斥

对立：互斥的一个特殊情况是“对立事件”，若A为一事件，则事件B={A不发生}，则称为A的对立事件或者说补事件。例：A={掷出奇数点}，B={掷出偶数点}，则A，B为对立事件，且必有一个发生

事件的和(或并)

C = A + B。若只要A发生或者B发生，或者两者同时发生，则C发生；若A，B都不发生，则C不发生。

例. A = {掷出偶数点} ={2,4,6} B = {掷出3的倍数} = {3,6}

C = A + B = {2,3,4,6}

概率的加法原理

若干个互斥事件之和的概率，等于各事件的概率之和。即$P(A_1+A_2+…) = P(A_1) +P(A_2)+…$ 该式称为加法定理。

若$A,\overline{A}$互为对立事件，则$P(\overline{A}) = 1 - P(A)$

事件的积

C = {A, B都发生}，记为AB。AB则由同属于这两个集合的那些试验结果组成，即两个集合的交叉。

多个事件$A_1,A_2…$的积的定义为$A = {A_1,A_2,…,都发生}$，$A = A_1 A_2…$或 $\prod_{i=1} ^n A_i$

事件的差

A-B={A发生，B不发生}

A-B：就是从构成A的那些试验结果中，去掉B内的那些。$A-B = A\overline{B}$

条件的概率

一般来讲，条件概率是在附加一定的条件下所计算的概率。

例如，A：掷出素数点；B：掷出奇数点；C掷出偶数点，A={2,3,5},B={1,3,5},C={2,4,6}

$P(A) = \frac{1}{2}$(在无条件下)

若加上“已知B发生”，则可能有3种情况：1，3，5，其中3，5代表A发生，因此A的条件概率为$P(A|B) = \frac{2}{3}$;同样在给定条件C发生的情况下，A的条件概率为$P(A|C) = \frac{1}{3}$

假设一试验有N个等可能的结果，事件A，B分别包含其中$M_1$个和$M_2$个结果，它们有$M_{12}$个是公共的。这就是事件AB所包含的试验结果数。若给定B发生，则我们考虑由起先的N个结果，局限到现在的$M_2$个，其中只有$M_{12}$个试验结果使得事件A发生，故$P(A|B) = \frac{M_{12}}{M_2}$。

$\frac{M_{12}}{M_2} = \frac{M_{12}/N}{M_2/N} = \frac{P(AB)}{P(B)}$

条件概率$P(A|B) = \frac{P(AB)}{P(B)}$

事件的独立性

设有两个事件A、B，A的无条件概率$P(A)$，与其在给定B发生之下的条件概率$P(A|B)$,一般是有差异的，这反映了两个事件之间，存在着一些联系。例如，$P(A|B)>P(A)$,则B的发生使A发生的可能性增大了，那么B促进了A的发生。

若$P(A) = P(A|B)$,则B的发生与否对A发生的可能性毫无影响，这时，在概率论上就称A，B两事件独立。$P(AB) = P(A)P(B)$, 该公式可以验证两事件是否独立。

推广到多个事件独立性的定义：设$A_1，A_2,….$为有限或无限个事件，如果从其中任意取出有限个$A_{i1},A_{i2},…A_{im}$都成立，$P(A_{i1},P_{i2},…P_{im} = P(A_{i1})P(A_{i2})…P(A_{im}))$,则称事件$A_1,A_2,…相互独立$。

同时，$P(A_{i1}|A_{i2},…A_{im}) = P(A_{i1})$,即任意事件发生的可能性大小，不受其他事件发生的影响。

乘法定理：

若干个独立事件$A_1,A_2,…A_n$之积的概率等于各事件概率的乘积。$P(A_1,A_2,…A_n) = P(A_1)…P(A_n)$

全概率公式

设$B_1,B_2…$为有限个或无限个事件，它们两两互斥，且在每次试验中至少一个发生。即$B_1+B_2+…= \Omega$，这样的一组事件称为“完备事件群”

现有任意一个事件A，有$A= A\Omega = AB_1 + AB_2 =…$,因为$B_1，B_2…$两两互斥，则$AB_1，AB_2$两两互斥，$P(A) = P(AB_1) + P(AB_2)+…$

由条件概率得：$P(AB_i) = P(B_i)P(A|B_i)$， 所以$P(A) = P(B_1)P(A|B_1) +P(B_2)P(A|B_2)+…$ 该式称为全概率公式

全部概率P(A)被分解成了许多部分之和。它的意义在于在比较复杂的情况下，直接算P(A)不容易，但A总是随某个事件$B_i$伴出，适当去构造$B_i$可以简化计算。

贝叶斯公式

在全概率公式的假定之下，有$P(B_i|A) = \frac{P(AB_i)}{P(A)} = \frac{P(B_i)P(A|B_i)}{\textstyle\sum_{i=1}P(B_i)P(A|B_i)}$

作用在于由结果推导原因。现有一个结果A发生了，在众多可能的原因中，到底是哪一个导致了这个结果。
