取样

發表於 2020-07-13   |   分類於 blog

本文重点：

  1. 总体
  2. 样本
  3. 如何设计取样
  4. 偏差的来源
  5. 取样的几种方式

背景：Mighty Gumball 是一家有名的糖果和巧克力制造商，他们想宣传一些他们的口香糖能保持多长时间的持久。问题是如何得到这些数据。

很明显，他们不能挨个放在嘴巴里去吃、去试，这样不仅会耗费掉所有的口香糖，造成巨大的金钱损失，还会损害员工的牙齿。那他们应该如何得到这部分数据呢？

这就引出了这篇文章的中心内容，取样，即测试一些口香糖样本，而不是所有的口香糖。

一、总体—population

统计意义上的总体是指我们试图测量、研究、分析的所有总体事物。例如，我们想进行人口普查，看看我们国家现在到底有多少人，这时候，总体就是国内全部的人。

二、样本—-sample

统计意义上的样本就是从总体中选取的部分。 我们可以从总体中选取样本来代表总体，因为这比较可行，耗费的时间、金钱、精力相对采用总体来说，也是非常少的，总而言之来说是可行的。

选取好的样本的关键是选择那些尽可能贴近总体的，这样样本所有具有代表性，这样样本与总体之间才有相似的特征。只有这样，我们才可以用样本来预测总体的一些特征。

三、如何设计取样

  1. 确定目标总体(target population)。首先要确定target population，这样才能知道样本是从哪里来的。通过target population，我们明确了要研究的事物。例如，我们想收集世界上所有的口香糖的数据，我们关心的是某种品牌的口香糖还是某种类型的口香糖，这个要弄清楚。
  2. 确定取样单位。在这个例子中，我们是想以每个口香糖为单位，还是以每盒口香糖为单位
  3. 定义取样框架。它基本上是一个列表，里面的每个样本都被命名或编号，你可以从中选择样本

四、偏差的来源

如果我们在进行取样时，没有客观公正的进行选择样本，而是有偏差的选择了一些样本，这样我们就会对总体产生错误的印象，进而做出错误的推断。一般来说，偏差的来源有这么几方面：

  1. 取样框架中一些样本被漏掉了，如果样本没有放在取样框架中，它们也不会进入你的样本中
  2. 不正确的取样单位。有时候以每个口香糖为单位是合适的，但有时候要考虑以每盒为单位
  3. 问卷调查中存在设计不好的问题。
  4. 取样时没有做的随机取样，而是以个人的主观判断去取样

五、几种取样方式

  1. 简单随机取样，从N个取样单位中随机选择n个取样单位，所有样本都有同样被选择到的可能性。简单随机取样又分为有放回和无放回。简单随机抽样可以通过抽签或者随机数生成器来实现。
  2. 分层抽样：把样本分成几个组，每个组都有类似的特征，然后从每个组中进行随机抽样。如口香糖可以按颜色分为黄色、绿色、红色、和粉色，每一种颜色叫作一层。
  3. 整群抽样：如口香糖可能分装成了一包一包的，可以随机选择几包，然后对每包里的口香糖进行试验
  4. 系统抽样：每隔几个样本选择一个样本。

一、离散变量与连续性变量

我们之前的例子中所涉及到的变量都是离散变量，例如滑雪成功的次数、答对的题目数、赢得的奖金数等等。离散变量都是确切的数字，他们在某种程度上都是可数的，像1，2，3…；我们之前所提到的概率计算都是针对这类变量的。

相反，连续性变量是不可数的，它们是用某种方式测量的，并且它们的取值是无限的；比如，它的取值可能是0与1之间的任意数字，要知道0与1的之间的数字是无限的。我们这篇文章就是要关注这类变量的概率分布。

二、概率密度函数

茱莉是个女学生，她想通过相亲的方式寻找她心目中的白马王子。她给自己定下来一条规则，就是“如果她的约会对象在20分钟之内没有出现，她将会离开”。

现在有一个问题：她想知道20分钟内，她等待超过5分钟的概率。

这个问题是求连续性变量的概率。为什么是连续的？因为约会对象可以在5分钟之内的任意一秒、任意一毫秒、任意一微妙内到达，5分钟内这个范围可以切割成无数，因而是不可数的，它是一个范围。因此，与离散变量针对某个特定值求概率不同的是，连续性变量是针对某个范围求概率。

这种情况下，我们可以使用概率密度函数来计算某一范围内连续变量的概率值，它告诉我们了概率分布是什么形状的。

四、正态分布扩展

  1. 若X ~ N(μx, σx 2)，Y ~ N(μy, σy 2)，则有X - Y ~ N(μ, σ2)，X+Y~ N(μ, σ2)

  2. 线性变换。若一个人的体重X服从正态分布，X ~ N(μx, σx 2)，试问这个人体重变为4X后服从什么分布。从X到4X的变换称为线性变换，线性变换改变的是最底层的值。线性变换后的正态分布aX + b ~ N(aμ + b, a2σ2)

  3. 独立观测值。若一个人的体重X服从正态分布，X ~ N(μx, σx 2)。若X1，X2，X3，X4是X的四个独立观测值，试问X1+X2+X3+X4服从怎么样的分布？X1 + X2 + … + Xn ~ N(nμ, nσ2)

  4. 使用正态分布代替泊松分布: X ~ Po(λ) 并且 λ > 15, 可以使用 X ~ N (λ, λ)来代替泊松分布；但这个时候会存在偏差，需要进行continuity correction调整，调整方法见下面。

  5. 使用正态分布代替二项分布：在泊松分布那篇文章中，提高了泊松分布在满足条件n>50,p<0.1的时候可以替代二项分布，以方便概率的计算。而正态分布在一定条件下也可以代替二项分布，这两个条件就是np > 5 并且 nq > 5。但通过正态分布计算的结果可能偏大，因为正态分布针对的是连续性变量，而二项分布针对的是离散变量。下面通过一个例子来说明这个问题。

以数字6为例，在我们将它转化为连续变量时，我们需要考虑它“附近范围“的数字，可能是从5.5到6.5这个范围内，这样一来，得到的概率值就大了。通过计算X<5.5的概率为0.3859，约等于0.387了。这说明我们不可以直接计算X<6,而需要对范围做一个调整

这个调整叫作continuity correction; 它根据不同的情况取舍不一样。

  • 当计算P(X ≤ a)时，我们要保证a值能够取到，所以要计算正态分布的P(X < a + 0.5)
  • 当计算P(X ≥ b)时，我们要保证b值能够取到，所以要计算正态分布的P(X > b - 0.5)
  • 当计算P(a ≤ X ≤ b)时，我们要a,b值同时能够取到，所以要计算正态分布的P(a - 0.5 < X < b + 0.5)
  • 当计算P(X<a)时，我们要保证a值取不到，所以要计算正态分布的P(X<a-0.5)
  • 当计算P(X>a)时，我们要保证a值取不到，所以要计算正态分布的P(X>a+0.5)

这三个条件看起来与几何分布很类似，那么两者的区别在哪里呢？

如果有固定数目的试验，并且我们想知道取得一定次数的成功的概率，那么就需要用二项分布；并且也可以通过二项分布知道大概可以成功多少次；如果我们想知道在取得第一次成功之前，需要进行多少次尝试，则需要使用几何分布。

三、泊松分布

我们来看这么一个场景：现在有一台爆米花机器，它每周都会出现故障，平均每周的故障次数为3.4次，试问它下周可以正常工作的概率有多大。

这个场景跟之前的几个不一样，因为没有一系列的试验。我们只知道故障发生的次数。我们大概只知道每周故障次数为3次或者4次，糟糕时，次数会更多；好的时候，没有故障。

这种情况所涉及到的概率分布称为Poisson distribution(泊松分布)

五、使用泊松分布代替二项分布

有时候，二项分布的概率比较难计算，比如当次数为3000时，我们就需要计算3000的阶乘，这是非常困难的。这个时候，我们可以使用泊松分布来大概的替代二项分布。

一、鸟儿分类系统

我们想打造一个鸟儿分类系统，来帮助我们识别不同种类的鸟儿。我们通过四个方面来识别一种鸟类，具体包括：weight(重量)，wingspan(翼展)，是否有webbed feet(蜘蛛网状的脚)，以及背部颜色。这几个方面的专业术语为features或attributes。

机器学习的一个任务就是classification(分类)。例如，在这个例子中，识别鸟儿种类的过程就是classification。所以，我们要选择相应的分类算法来处理这个任务。

为了训练我们的算法，我们需要喂给它数据，这些数据又可以称为training set(训练集)。在我们的这个例子中，我们有六个training examples，如上表所示。每个training example有四个features和一个*target variable。这个时候我们告诉算法每个feature和他们对应的target variable, 我们分类算法的目的就是找出features和target variable之间的关系。然后使用*test set(测试集)来对我们的算法进行测试，这时候我们不会告诉算法target variable，而是由算法自己决定每个example属于哪一个分类。

二、机器学习的两类任务

在上面的例子中，我们其实接触到了机器学习第一种任务：classfication(分类)，在classification中，我们的工作是预测每个instance将会落入哪一个分类中。另外一项任务是regression(回归)。回归是用来预测一个numeric value(数值)。classification和regression都属于supervised learning(监督学习)，因为我们告诉了算法要去预测什么。

与supervised learning相对的是unsupervised learning(无监督学习)。在无监督学习中，给定的数据都是没有标签和目标值的。无监督学习的一类任务是clustering(聚类); 在无监督学习中，有时候我们想要找到一些统计值来描述数据，这个过程叫作density estimation(密度估计)。无监督学习的另外一项任务是减少features的数量，然后再把它们用两个到三个维度进行可视化。

四种任务所用到的算法

如何决定采用哪个算法？

如果你想预测某个target value(目标值)，这时候就用到了监督学习。如果target value像Yes/No, 1/2/3, A/B/C, or Red/Yellow/Black这种，就应该使用classification；如果target value属于0.00 to 100.00, or -999 to 999, 负无穷到正无穷，这时候就要采用regression;

如果你不是预测一个target value，这时候就要用到unsuper- vised learning. 例如，如果想将数据划分成几个组，这个时候就需要用到clustering；如果想要衡量每个数据与其分组的符合情况，这个时候就需要用到density estimation

三、机器学习步骤

  2. 准备input data：一旦我们有了数据，我们要确保它以一种可用的形式存在。标准形式的优点在于，我们可以混合、匹配算法和数据
  3. 分析input data：通过观察数据，我们或许能识别出某些模式，或者能辨别出某些数据与其他数据有极大的不同；将数据用一维、二维甚至三维的方式进行展示，也非常有帮助；
  4. 训练算法：这个步骤是机器学习发生的地方。这个步骤和接下来的步骤是算法的核心所在。你用第一二步所得到的干净数据来训练算法，然后从中提取信息。要注意，在无监督学习中，没有训练这一步，因为没有target value。

一、Range

range(范围)描述了一组数据展开情况，通过Upper bound(上界,最大值)与Lower bound(下界,最小值)的差值，得出range.

⚠️：range有它的局限性，例如它只告诉了我们数据的“宽度”，它并没有告诉我们数据是如何分布的；并且它很容易受到离群值的影响，因为它的计算方式是最大值减去最小值，所以有时候它并不能告诉我们数据的真实面貌。

二、interquartile range(四分位间距)

既然Range的局限性在于离群值的影响，那么是否可以规避这些离群值？这个方法就是只使用数据中心及其周围的数据值。我们可以将数据按照从小到大的顺序排列，然后将这些数据分成数量相等的四块，每一块包含四分之一的数据。如下图所示：

最小的四分位数称为lower quartile，或者是first quartile(Q1)；最大的四分位数称为upper quartile或Q3. 中间的四分位数称为Q2，因为它把数据分成了两部分。这两部分数据的范围称为interquartile range.

四分位数是如何规避离群值影响的？

首先，uppper和lower quartiles是位置固定的，所以有25%的数据是小于lower quartile的，同理，有25%的数据是大于upper quartile。这意味着interquartile range只使用了中间50%的数据，所以离群值被抛弃了。

三、percentiles

与四分位数类似，我们可以把数据分成若干等份。例如，我们可以把数据分成10份，每份包含十分之一的数据。把数据分割的数值称为百分位数，如我们把数据分成了10份，所以这个值称为是分位数。我们把使用这种方式构成的范围称为interpercentile range。每个百分位数是通过它分割数据的百分比来表示的。

相关分析：

  • 无论是在数据科学还是研究中，很多建模项目的探索性数据分析都要检查预测因子之间的相关性，以及预测因子和目标量之间的相关性。给定变量X和Y，它们均有测量数据。如果变量X的高值随变量Y的高值而变化，并且X的低值随Y的低值的变化而变化，那么我们称X和Y是正相关的。如果X的高值随Y的低值的变化而变化，那么我们称变量X和Y是负相关的。
  • 衡量事物之间或称变量之间线性相关程度的强弱，并用适当的统计指标表示出来的过程
  • 比如家庭收入和支出、一个人所受教育程度与其收入、子女身高和父母身高等

相关系数：

  • 一种用于测量数值变量间相关程度的度量，取值范围在-1到+1之间
  • 相关系数r的正负号表示变化方向。“+”号表示变化方向一致，即正相关；“-”号表示变化方向相反，即负相关
  • r的绝对值表示变量之间的密切程度(即强度)。绝对值越接近1，表示两个变量之间关系越密切；越接近0，表示两个变量之间关系越不密切
  • 相关系数的值，仅仅是一个比值。它不是由相等单位度量而来(即不等距)，也不是百分比，因此，不能直接作加、减、乘、除运算
  • 相关系数只能描述两个变量之间的变化方向及密切程度，并不能揭示两者之间的内在本质联系，即存在相关的两个变量，不一定存在因果关系

连续变量的相关分析：\连续变量即数据变量,它的取值之间可以比较大小，可以用加减法计算出差异的大小，如年龄、收入、成绩等变量；当两个变量都是正态连续变量，而且两者之间呈线性关系时，通常用Pearson相关数来衡量。

协方差：协方差是一个反映两个随机变量相关程度的指标，如果一个变量跟随着另一个变量同时变大或变小，那么这两个变量的协方差就是正值。

虽然协方差能反映两个随机变量的相关程度(协方差大于0的时候表示两者正相关，小于0的时候表示两者负相关)，但是协方差值的大小并不能很好地度量两个随机变量的关联程度。这是因为在二维空间中分布着一些数据，我们想知道数据点坐标X轴和Y轴的相关程度，如果X与Y的相关程度较小但是数据分布的比较离散，这样会导致求出的协方差值较大，用这个值来度量相关程度是不合理的。

为了更好的衡量两个随机变量的相关程度，引入了Pearson相关系数，其在协方差的基础上除以了两个随机变量的标准差。

Pearson是一个介于-1和1之间的值，当两个变量的线性关系增强时，相关系数趋于1或-1；当一个变量增大时，另一个变量也增大时，表明它们之间是正相关的，相关系数大于0；如果一个变量增大，另一个变量却减小，表明它们之间是负相关的，相关系数小于0；如果相关系数等于0，表明它们之间不存在线性相关关系。

方差分析

發表於 2020-06-27   |   分類於 blog

方差分析：检验多个总体均值是否相等，通过分析观察数据的误差判断总体均值是否相等。

使用场景：需要对比多个包含数值型数据的组，比如A，B，C，D；方差分析是一种检验多个组之间统计显著性差异的统计学方法。

基本思想：比较两类误差，以检验均值是否相等，比较的基础是方差比；如果系统误差显著地不同于随机误差，则均值就是不想等的；反之，均值就是相等的。

结合实例理解相关概念：

  • 因素或因子(factor):所要检验的对象，比如在这个例子中，要分析行业对投诉次数是否有影响，行业是要检验的因素或因子
  • 水平或处理(treatment):因素的不同表现，即每个自变量的不同取值称为因素的水平
  • 观察值：在每个因素水平下得到的样本值，每个行业被投诉的次数就是观察值
  • 试验: 这里只涉及一个因素，因此称为单因素四水平的试验
  • 总体：因素的每一个水平可以看作是一个总体，比如零售业、旅游业、航空公司、家电制造业可以看作是四个总体
  • 样本数据: 被投诉次数可以看作是从这四个总体中抽取的样本数据
  • 随机误差: 因素的同一水平或总体下，样本各观察值之间的差异，比如，同一行业下不同企业被投诉次数是不同的，这种差异可以看成是随机因素的影响，称为随机误差
  • 系统误差：因素的不同水平或不同总体下，各观察值之间的差异，比如不同行业之间的被投诉次数之间的差异，这种差异可能是由于抽样的随机性所造成的，也可能是由于行业本身所造成的，后者所形成的误差是由系统性因素造成的，称为系统误差
  • 组内方差: 因素的同一水平或同一个总体下样本数据的方差，比如零售业被投诉次数的方差，组内方差只包含随机误差
  • 组间方差：因素的不同水平或不同总体下各样本之间的方差；比如四个行业被投诉次数之间的方差，组间方差包括随机误差，也包括系统误差

方差的比较：

  • 若不同行业对投诉次数没有影响，则组间误差中只包含随机误差，没有系统误差。这时，组间误差与组内误差经过平均后的数值就应该很接近，它们的比值就会接近1
  • 若不同行业对投诉次数有影响，在组间误差中除了包含随机误差外，还会包含有系统误差，这时组间误差平均后的数值就会大于组内误差平均后的数值，它们之间的比值就会大于1
  • 这个比值大到某种程度时，就可以说不同水平之间存在显著差异，也就是自变量对因变量有影响
  • 判断行业对投诉次数是否有显著影响，实际上也就是检验被投诉次数的差异主要是由于什么原因所引起的。如果这种差异主要是系统误差，说明不同行业对被投诉次数有显著影响

单因素方差分析：模型中有一个自变量(因素)和一个观测变量，其实就是在一个因素的不同水平下，观测变量均值差异的显著性检验。

提出假设：

  • H0: u1 = u2 = … = uk, 自变量对因变量没有显著影响
  • H1: u1, u2…u4不完全相等，自变量对因变量有显著影响
  • 拒绝原假设，只表明至少有两个总体的均值不相等，并不意味着所有的均值都不相等。

  • 水平的均值

  • 全部观察值的总均值

  • 误差平方和

  • 均方(MS)

    水平的均值：假定从第i个总体中抽取一个容量为ni的简单随机样本，第i个总体的样本均值为该样本的全部观察值总和除以观察值的个数

全部观察值的总均值：全部观察值的总和除以观察值的总个数

水平项平方和SSA：各组平均值与总平均值的离差平方和，反映各总体的样本均值之间的离异程度，又称组建平方和，该平方和既包括随机误差，也包括系统误差。

假设检验

發表於 2020-06-27   |   分類於 blog

假设：对总体的具体数值(如均值、比例等)所作的陈述，比如新的配方的药效要比原来的更好；推广新的教育方案后，教学效果有所提高；醉驾判定为刑事犯罪后使得交通事故减少；男生和女生在选文理科时存在性别因素影响。

假设检验(也称显著性检验)是对总体数据提出某种假设，然后再利用样本的信息(数值信息)判断假设是否成立的过程。例如，假设“醉驾判定为刑事犯罪后使得交通事故减少”，我们要检验这个假设收否成立。假设检验的目的是确定一个观测到的效果是否是由随机性造成的。

为什么需要假设检验？我们为什么需要做出一个假设？为什么不能只是查看实验的结果，然后选择处理结果更好的那一组？问题的答案在于，人们在思想上倾向于低估天然随机行为的范围。一个典型的例证就是难以预料的极端事件，即黑天鹅事件；另一个例证就是人们倾向于将随机事件曲解为具有某种显著性的模式；为此，人们提出了统计假设检验方法，目的是使研究人员免受随机性的愚弄。统计假设试验是对任何随机试验的进一步分析，意在评估随机性是否可以合理地解释A组和B组之间观测到的差异。

假设检验的步骤可以分为：提出假设、确定适当的检验统计量(衡量假设是否成立的标准)、规定显著性水平、计算检验统计量的值、做出统计决策。

显著性水平是一个概率值，是指可接受的“异常值“的概率。假设某工厂生产一批零件，允许一百件中有两件次品，那么显著性水平就是2%。比较专业的定义是原假设为真时，拒绝原假设的概率，表示为alpha，常用取值为0.01，0.05，0.10。统计学家引入了统计显著性的概念，用于衡量一个实验，所生成的结果是否会比随机情况下可能给出的结果更极端。如果生成的结果超出了随机变异的范围，则我们称它是统计显著的

检验统计量：计算检验的统计量(用于检验处理效果的度量)，根据给定的显著性水平，查表得出相应的临界值，将检验统计量的值与显著性水平的临界值进行比较，得出拒绝或不拒绝原假设的结论。

其他概念：

  1. 原假设与备择假设：待检验的假设又叫原假设，也可以叫零假设，表示为H0，零假设表示原假设没有差异，没有改变。如醉驾判定为刑事犯罪后对交通事故减少没有影响。与原假设对比的假设叫作备择假设，表示为H1,如醉驾判定为刑事犯罪后使得交通事故减少。
  2. 备择假设：与原假设对比的假设叫做备择假设，表示为H1。总而言之,零假设和备择假设必须涵盖了所有的可能性
  3. 小概率：在一次试验中，一个几乎不可能发生的事件发生的概率；在一次试验中，小概率事件一旦发生，我们就有理由拒绝原假设。
  4. P值：是一个概率值。如果原假设为真，P值是抽样分布中大于或小于样本统计量的概率(即没有落在拒绝域中)；左侧检验时，P值为曲线上方小于等于检验统计量部分的面积；右侧检验时，P值为曲线上方大于等于检验统计量部分的面积；对于一个加入了零假设的偶然性模型，p值指得到与观测结果一样不寻常或极端的结果的概率。在衡量统计显著性时，简单地查看绘图并不是一个非常精确的方法，人们更关注的是P值。P值表示随机模型生成的结果比观测结果更极端的频数。P值是结果由随机所导致的概率，P值越大，说明结果随机的可能性越大，越支持H0，否则，则拒绝H0。P值概念结合图比较好理解，参考下图。

在我们拿到数据后，需要对数据的样貌或概况(比如数据分布情况、均值、中位数、众数)有所了解，这个过程叫作描述统计。“描述统计是通过图表或数学方法，对数据资料进行整理、分析，并对数据的分布状态、数字特征和随机变量之间关系进行估计和描述的方法”，通过描述统计，可以让我们对数据有初步的了解，并且很有可能得出初步的结论并提出假设。

第一个比较有用的图表是histogram——直方图。它的每一直条的面积等于该段范围的频率。它的计算方法不再赘述，重点是我们能从图中得到什么信息。

直方图可以告诉我们数据的两个重要特性：集中趋势(即越靠近中央部分，频率越高)和离中趋势(绝大多数个体值并不与平均水平完全重合，而是不同程度地偏离平局水平)。根据集中和离散的程度，数据分布类型可以分为对称分布和偏态分布(右偏态分布和左偏态分布)。

区分左右偏态分布的方法很简单，看尾巴在哪边就是什么偏态分布。导致分布偏离中央的数据称为outliers(离群值)，这个离群值很重要，因为它决定了average的选择，比如选择用mean(平均数)、median(中位数)、还是mode(众数)。
